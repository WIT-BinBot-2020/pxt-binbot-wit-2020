# pxt-binbot-wit

A MakeCode extensions for WITs Binbot Project



## Usage


### Setting the Radio Group

```blocks
radio.setGroup(187)
```

### Moving Binbot

```blocks
input.onButtonPressed(Button.A, function () {
    Binbot.moveBinbot(100, 0, 0)
})
```
